 📊 Superstore SQL Sales Analysis Project

An end-to-end SQL project analyzing retail sales data to generate business insights using MySQL Workbench.


 🎯 Objective

To analyze sales performance, profitability, and regional trends to identify key business insights and decision-making opportunities.


 📂 Dataset

Sample Superstore Dataset

- Records: 9,994 transactions
- Fields include:
  - Sales
  - Profit
  - Quantity
  - Discount
  - Category & Sub-Category
  - Region & State
  - Shipping details


 🛠 Tools Used

- MySQL Workbench
- SQL


 SQL Concepts Used

- SELECT
- DISTINCT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions
- Window Functions (ROW_NUMBER)
- Data Cleaning
- Data Validation
- Exploratory Data Analysis (EDA)


 Key Insights

- Technology is the highest revenue and profit-generating category.
- Furniture shows lower profitability despite high sales.
- West region is the strongest performing region.
- South region is the weakest performing region.
- California and New York are top profit contributors.
- Several states (e.g., Texas, Ohio) are loss-making.


 Key Outcome

This project demonstrates how SQL can be used to transform raw retail data into actionable business insights for decision-making.
